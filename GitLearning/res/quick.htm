Git in Windows

Download

Make Your Identity

git config --global user.name/email Set your user name and email address for projects globally.

Set SSH Key

ssh-keygen -t rsa -C 'your email' Generate SSH key then place id_rsa.pub into github SSHKeys

Create New Repository

git init [proj-name] Initialized empty git repository.

Add a file

echo 'Hello Git' > welcome.txt Create a new file and set some data-info.
git add welcome.txt put the new file into the staging Area.
git commit -m 'Init git project' Submit to the repository with operation tips.

Push to Remote Repository

git remote add origin [proj-url] Associate to the remote repository.
git push -f origin master Force to push all the files of this new project.
